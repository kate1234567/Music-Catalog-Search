# Music Catalog Search

## Описание

Music Catalog Search — приложение на C#, реализующее каталог музыки с возможностью поиска по артистам, альбомам, сборникам и трекам.

В проекте спроектирована объектная модель музыкального каталога: артисты, альбомы, жанры, треки и плейлисты. Также реализован небольшой поисковый движок для работы с данными каталога.

Проект выполнен в рамках лабораторной работы по дисциплине «Объектно-ориентированное программирование».

## Используемые технологии

- C#
- .NET
- Windows Forms

## Основные возможности

- хранение информации об артистах;
- хранение информации об альбомах и треках;
- работа с музыкальными жанрами;
- создание плейлистов и сборников;
- поиск артистов, альбомов и треков;
- использование паттернов проектирования.

## Использованные паттерны

- Builder;
- Facade.

## Использованные принципы ООП

- инкапсуляция;
- наследование;
- композиция;
- работа с интерфейсами и классами сущностей.

## Запуск проекта

1. Открыть решение `CatalogMusic.sln` в Visual Studio.
2. Собрать проект.
